About Risk Scoring

What is Metadata Risk Scoring?

This tool helps you assess the risk level of your metadata based on several key factors:

Risk Factors:

  • Metadata Type: Different types of metadata have varying levels of sensitivity and importance
  • Data Privacy Category: Higher privacy categories indicate more sensitive data
  • Dependencies: More dependencies increase risk as changes affect more systems
  • Documentation: Better documentation reduces risk by improving understanding

Risk Levels:

  • Low Risk: Well-documented, low-dependency metadata
  • Medium Risk: Moderate complexity and dependencies
  • High Risk: High-dependency, sensitive metadata with limited documentation

How to Use:

  1. Select your metadata type from the dropdown
  2. Choose the appropriate data privacy category
  3. Enter the number of dependencies (systems that use this metadata)
  4. Specify the amount of documentation available
  5. Click "Confirm" to calculate your risk score

Please enter your metadata information below, and press confirm when you are ready.

Quick Reference:

Metadata Type: Choose the category that best describes your metadata
Data Privacy: Select based on the sensitivity of the data
Dependencies: Count how many systems or processes use this metadata
Documentation: Number of documentation items (guides, specs, etc.)
